Fórmulas Matemáticas

Regresión Logística

P(Y=1|X) = 1 / (1 + e^(-(β₀ + β₁X₁ + β₂X₂ + ... + βₚXₚ)))

Regresión Lineal

Y = β₀ + β₁X₁ + β₂X₂ + β₃X₃ + ε

Regresión Polinomial

Y = β₀ + Σ(βᵢ×Xᵢ) + Σ(βᵢⱼ×XᵢXⱼ) + Σ(βᵢⱼₖ×XᵢXⱼXₖ) + ... + ε
